# Student Registration Confirmation System

## Features

- Bootstrap 5 UI
- PHP Form Handling
- POST Method
- Form Validation
- PHP Functions
- Dynamic Grade Calculation
- Greeting Based on Time
- Current Date
- Header & Footer using include()
- Responsive Design
- Font Awesome Icons

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- PHP
- XAMPP

## How to Run

1. Copy the project folder into `htdocs`.
2. Start Apache from XAMPP.
3. Open:
   http://localhost/Student-Registration-System/
4. Fill the form and submit.
